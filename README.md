# Latteria Sociale di Fondo — Website Redesign

> Editorial prototype + commercial proposal prepared by **Flareo** for Latteria Sociale di Fondo — a cooperative cheesemaker in Val di Non, Trentino (founded 1845).

## Live prototype
https://flareo-agency.github.io/latteria-sociale-fondo/

## Documents
- [Proposta commerciale](./proposta.html) — client-ready editorial proposal (HTML)
- [Audit completo](./audit-report.md) — full website audit (18 pages scraped, 14 critical issues)

## Scope
Single-file editorial website inspired by ginivini.com/creazioni — graduating product panels, Fraunces display serif, 20 products grouped into 4 chapters (Nostrani, Caciotte, Specialità, Yogurt & Panna), dedicated Monte Mais Kofel editorial, territory + visita + punti vendita + contatti. E-commerce excluded from v1 by client request.

## Stack
- Single-file HTML/CSS/JS
- Fraunces (display) + Inter (body), Google Fonts
- IntersectionObserver scroll reveals
- Client-side form validation
- IT/DE/EN language switcher

## Investment
€ 4–6k Base (una tantum, prezzo fisso) · +€1,8k Premium (CMS + multilingua) · Enterprise su misura · 7–9 settimane.
